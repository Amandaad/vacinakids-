# VacinaKids

Projeto Ionic + Angular para acompanhamento vacinal infantil.

## Funcionalidades
- Cadastro de crianças
- Carteira digital de vacinação
- Histórico vacinal
- Campanhas de vacinação
- Indicadores de vacinas pendentes e atrasadas
